h1. SlidesJS version 3.0.x beta

SlidesJS, a responsive, CSS3 and touch based slideshow plugin for jQuery. Packed with a useful set of features to help novice and advanced developers alike create elegant and user-friendly slideshows.

SlidesJS is provided with limited support but if you find a bug please submit an "Issue":https://github.com/nathansearles/slides/issues with a brief description of the bug and a link to where it can be reviewed.

**SlidesJS version 3.0.x beta is not meant for production deployment.**

For the current production ready version of SlidesJS visit "http://slidesjs.com/":http://slidesjs.com/.
